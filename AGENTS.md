# AGENTS.md

Agent configuration for the `outboardphp/outboard` monorepo.

## Agent skills

### Issue tracker

Issues are tracked in GitHub Issues on `outboardphp/outboard` via the GitHub MCP server. See `docs/agents/issue-tracker.md`.

### Triage labels

Uses the default five-role label vocabulary (needs-triage, needs-info, ready-for-agent, ready-for-human, wontfix). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context repo; domain doc lives at `docs/architecture.md` (not `CONTEXT.md`). See `docs/agents/domain.md`.
